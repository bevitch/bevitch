# Styles

## Background color

You can use following to set element color:

- use `.ne--color-(color)` (for example `.ne-color-red`) to set element background,
- use `.ne--color-(color)` and `.ne--darken-(1, 2 or 3)` (for example `.ne--darken-2`) to darken selected color,
- use `.ne--color-(color)` and `.ne--lighten-(1, 2 or 3)` (for example `.ne--lighten-2`) to lighten selected color,

<Preview>
  <Story name="Background color">
    {{
      components: { NeApplication },
      data() {
        return {
          colors: [
            'grey',
            'red',
            'green',
            'green-light',
            'blue',
            'blue-indigo',
            'yellow',
            'yellow-amber',
            'primary',
            'secondary',
            'success',
            'info',
            'warning',
            'error',
          ]
        }
      },
      methods: {
        getColor(prefix, color) {
          return `${prefix}${color}`;
        },
        getShade(prefix, shade) {
          const suffix = shade < 4 ? `lighten${shade - 4}` : shade === 4 ? '' : `darken-${shade - 4}`;
          return suffix.length > 0 ? `${prefix}${suffix}` : '';
        }
      },
      template: `
        <ne-application>
          <div style="display: flex;flex-wrap: wrap;">
            <div v-for="(color, index) in colors" :key="index" style="width: 300px;">
              <div v-for="i in 7" :key="i" :class="[getColor('ne--color-', color), getShade('ne--', i)]">{{ getColor('.ne--color-', color) }} {{ getShade('.ne--', i) }}</div>
            </div>
          </div>
        </ne-application>`,
    }}
  </Story>
</Preview>

## Typography

You can use following classes to standarize headers appearance:

```
.ne--display-4
.ne--display-3
.ne--display-2
.ne--display-1
.ne--headline
.ne--title
.ne--subtitle-1
.ne--subtitle-2
.ne--body-1
.ne--body-2
.ne--caption
.ne--overline
```

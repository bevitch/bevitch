# Getting started

## How to import directive

If you want to import directive in your component use following syntax:

```ts
import { ClickOutside } from '@bevitch/core';
```

and declare these components in your app:

```ts
export default {
  //...
  directives: {
    ClickOutside
  },
  //...
}
```

Then you can use these components in your template:

```html
<template>
  <div v-click-outside="... (some method)" style="background-color: #000; color: #fff;">
    Click outside of me and check console
  </div>
</template>
```

## TypeScript support

If you are using TypeScript in your app, you must declare module for this library, for example:

**/src/@types/bevitch.d.ts**:
```ts
declare module '@bevitch/core';
```

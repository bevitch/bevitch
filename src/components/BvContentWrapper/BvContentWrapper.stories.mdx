# Button

<Props of={BvContentWrapper} />

Default layout breakpoints are:
* xs: 320px
* sm: 500px
* md: 768px
* lg: 1200px

On xs and sm content is 100% wide.
On md content is 768px wide.
On lg content is 1200px wide.

Sugested coulmn layout options:

xs - four 60px columns with margin 10px:
```
|<--------------------------------------------320px--------------------------------------------->|
|                                                                                                |
|<-10px->|<---60px--->|<-20px->|<---60px--->|<-20px->|<---60px--->|<-20px->|<---60px--->|<-10px->|
```
md - eight 76px columns with margin 10px::
```
|<--------------------------------------------768px------------------------------------------------------------------------------------------------------------------------------------->|
|                                                                                                                                                                                        |
|<-10px->|<---76px--->|<-20px->|<---76px--->|<-20px->|<---76px--->|<-20px->|<---76px--->|<-20px->|<---76px--->|<-20px->|<---76px--->|<-20px->|<---76px--->|<-20px->|<---76px--->|<-10px->|
```
lg - twelve 80px columns with margin 10px:
```
|<--------------------------------------------1200px---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------->|
|                                                                                                                                                                                                                                                                                |
|<-10px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-20px->|<---80px--->|<-10px->|
```
